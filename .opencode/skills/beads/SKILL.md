# Beads Workflow Skill

A skill for managing issue tracking workflow using the beads system.

## Overview

This skill provides a structured approach to issue tracking and session management using the beads system (beads_viewer).

## Key Concepts

- **One task per session**: Focus on completing one beads issue per session
- **Execution tracking**: Keep track of progress and working memory
- **Session protocols**: Clear start and end procedures

## Usage

1. At session start: Load the skill and follow session-start template
2. During work: Use working-memory notes to track progress
3. At session end: Follow session-close template
