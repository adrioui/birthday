# Opencode Skills

This directory contains skill definitions for working with opencode agents.

## Skills

### beads

Issue tracking workflow using the beads system.

- **Focus**: One task per session, execution tracking
- **Templates**: Session start/close, working memory notes
- **References**: Execution vs planning, one-task-per-session, working memory notes

### agent-browser

Discovery and implementation workflow with browser-based verification.

- **Focus**: Pattern discovery, implementation with verification
- **Templates**: Discovery session, implementation session, handoff notes
- **References**: Discovery vs implementation, progressive disclosure, snapshot references

## Usage

Skills are loaded when needed to provide structure and guidance for specific workflows.

For more information, see AGENTS.md for agent workflow guidelines.
